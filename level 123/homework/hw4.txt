4) ახსენით რა არის http response და response codes; ჩამოთვალეთ და განმარტეთ ნასწავლი კოდები (ჩამოწერეთ txt ფაილში) 

HTTP Response — ეს არის სერვერის პასუხი იმ HTTP Request-ზე, რაც კლიენტმა გაუგზავნა.

HTTP Response Codes — ეს არის რიცხვითი კოდები, რომლებიც გვიჩვენებს, როგორ დაამუშავა სერვერმა მოთხოვნა.
