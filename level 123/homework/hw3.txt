3) ახსენით რა არის http request და request methods; ჩამოთვალეთ და განმარტეთ ნასწავლი მეთოდები (ჩამოწერეთ txt ფაილში) 

HTTP Request — ეს არის მოთხოვნა, რომელსაც ვებ-ბრაუზერი აგზავნის ვებ-სერვერზე, რათა მიიღოს ან გააგზავნოს ინფორმაცია.

HTTP Method — ეს არის მოქმედების ტიპი, რომელიც განსაზღვრავს, რას აკეთებს კლიენტი (მაგ. ვებ-ბრაუზერი) სერვერზე გაგზავნილი მოთხოვნით (HTTP Request).
